# opennebula-users

Scripts para crear y borrar usuarios en OpenNebula.

## Prerrequisitos

```shell
pip3 install click
```

## Funcionamiento

Crear usuarios:

```shell
python3 crear_usuarios.py
```

Borrar usuarios:

```shell
python3 borrar_usuarios.py
```
